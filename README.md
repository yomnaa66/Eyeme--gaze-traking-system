# Eyeme--gaze-traking-system
This graduation project was built to attempt to solve the problem of poeple with disabilities like quadriplegia who can not control devices. We tried to provide an affordable, highly accurate device for the majority of them to use. This is the AI part of the project-we used Facial Landmark Detection Model and Gaze Estimation Model (L2CSNet_Gaze360)

EyeMe: Eye-Tracking Assistive Technology
EyeMe is an AI-powered system designed to assist individuals with physical disabilities. It enables users to interact with and control electronic devices solely through eye movements and gaze estimation.

Features
Real-time Gaze Tracking: High-accuracy pupil detection and movement analysis.

Device Control: Translates eye movements into commands for computers and other devices.

Multi-Platform Integration: Includes a mobile application for configuration and remote control.

Assistive UI: Designed to minimize eye strain while maintaining precise control.

Technical Architecture
The system integrates Computer Vision, Deep Learning, and Embedded Systems.

AI Models: Utilizes L2CSNet trained on the Gaze360 dataset for robust gaze estimation across different head poses.

Libraries: Developed using Python, OpenCV, and Dlib for facial landmark detection and real-time image processing.

Hardware: Integrated with embedded systems to facilitate physical device interaction.
